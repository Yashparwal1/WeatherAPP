## Create Your API KEY and Run the Web App
```
1. Go to https://rapidapi.com/ and signup
2. Search for weather by API Ninja or visit https://rapidapi.com/apininjas/api/weather-by-api-ninjas
3. Subscribe for free plan (3000 Requests per month)
4. Now you will get you API key, paste it in the JS file as the vaue of 'X-RapidAPI-Key' header.
5. "HURREY!!" Now just run the HTML file and you are on the go...
```
